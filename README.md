# no-build-break-plugin

用于禁止玩家在服务器内放置和拆除方块。

## 功能

- 禁止玩家放置方块
- 禁止玩家拆除方块
- 适合用于大厅、展示区或纯浏览型地图

## 适用场景

- 大厅服
- 跳板服
- 观战区
- 不希望玩家破坏地图的玩法区域
